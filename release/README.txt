Radiant Core Windows Executables
=================================

These are the compiled Windows executables for Radiant Core v2.0.0.

Files Included:
--------------
- radiantd.exe     - Main Radiant Core daemon (12.8 MB)
- radiant-cli.exe  - RPC client utility (4.2 MB)  
- radiant-tx.exe  - Transaction utility (5.5 MB)

Quick Start:
-----------
1. Run the daemon:
   radiantd.exe

2. Check daemon status:
   radiant-cli.exe getblockchaininfo

3. Create transactions:
   radiant-tx.exe --help

System Requirements:
-------------------
- Windows 10/11 (64-bit)
- No additional dependencies required

Build Information:
-----------------
- Built with MinGW-w64 GCC
- Portable Windows build system
- Version: v2.0.0-7cfb963

For more information, visit: https://radiantblockchain.org
